* Movie Portal

 1. Project Type
Responsive Frontend Movie Streaming Web Application (OTT Platform Clone) built using HTML, CSS, and JavaScript.

---

 2. Description

This is a fully functional movie streaming web application inspired by platforms like Zee5. It allows users to browse, search, and watch movies with a smooth and responsive user interface.

The project is built using pure frontend technologies without any frameworks, focusing on strong fundamentals of web development.

---

3. Features

- Browse movies with a clean and modern UI  
- Search movies dynamically in real-time  
- Watch movies using a built-in video player  
- Navigation handled using localStorage  
- Multiple pages (Home, Watch, Help Center)  
- Fully responsive design for mobile devices  

---

4. Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Local Storage API

6. How It Works

- Movie data is handled using JavaScript  
- Search filters movies instantly based on user input  
- When a movie is clicked, its data is stored in localStorage  
- The watch page retrieves and plays the selected movie
